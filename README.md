## 📖 About The Project

A **Production-Ready Full Stack Gen AI Web Application** built to simulate a real-world job preparation system — from authentication and resume analysis to AI-powered interview generation.

---

## ✨ Features

* 🔐 Secure Authentication (JWT + HTTP-only Cookies)
* 🚫 Token Blacklisting for secure logout
* 📄 Resume Upload & Parsing
* 🤖 AI-Based Interview Question Generation
* 🧠 Skill Gap Detection using AI
* 📑 ATS-Optimized Resume Creation
* 🖨 Dynamic PDF Generation using Puppeteer
* ⚡ Fully Responsive UI

---

## 🛠 Tech Stack

* **Frontend:** React.js, Tailwind CSS, Context API
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT + Token Blacklisting
* **AI:** Gemini API
* **PDF Generation:** Puppeteer

```

```

